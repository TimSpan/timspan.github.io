# router-link

  1. vue-router 提供了一个全局组件 router-link
  2. router-link 实质上最终会渲染成a链接 to属性等价于提供 href属性(to无需#)
  3. router-link 提供了声明式导航高亮的功能(自带类名)
<router-link to="/">Go to Foo</router-link>
<router-link to="/find">Go to Bar</router-link>
<a href="#/find">朋友</a>

# 声明式导航

在跳转路由时, 可以给路由对应的组件内传值

在router-link上的to属性传值, 语法格式如下:

  • /path?参数名=值
  • /path/值 – 需要路由对象提前配置 path: “/path/参数名”

对应页面组件的接收传递过来的值

# 重定向

匹配path后,强制跳转path路径

网页打开url默认hash值是/路径

redirect是设置要重定向到哪个路由路径

path: '/',
redirect: '/layout'

path: '/layout',
redirect: '/layout/home',//重定向

404

路由最后, path匹配*(任意路径) – 前面不匹配就命中最后这个

# 模式设置

  • hash路由例如: http://localhost:8080/#/home
  • history路由例如: http://localhost:8080/home (以后上线需要服务器端支持, 否则找的是文件夹)

# 嵌套路由

在现有的一级路由下, 再嵌套二级路由

// main.js

import Vue from 'vue'
import App from './App.vue'
import My from './view/My.vue'
import Find from './view/Find.vue'
import VueRouter from 'vue-router'
import Ranking from './view/Second/Ranking.vue'
import Recommend from './view/Second/Recommend.vue'
import SongList from './view/Second/SongList.vue'

Vue.use(VueRouter)
//每个路由应该映射一个组件
const routes = [
  { path: '/', component: My },
  //根路径对应的就是My组件
  {
    path: '/find', component: Find, children: [
      {
        path: 'ranking',
        component:Ranking
      },
      {
        path: 'recommend',
        component:Recommend
      },
      {
        path: 'songlist',
        component:SongList
      },
    ]
  }
]
// 创建 router 实例，然后传 `routes` 配置
const router = new VueRouter({
  routes, // （缩写）相当于 routes: routes
  mode:"history"
})
new Vue({
  // 记得要通过 router 配置参数注入路由，从而让整个应用都有路由功能
  router,
  render: h => h(App)
}).$mount('#app')


App.vue

<template>
  <div>
    <h1>Hello App!</h1>
    <p>
      <!-- 使用 router-link 组件来导航. -->
      <!-- 通过传入 `to` 属性指定链接. -->
      <!-- <router-link> 默认会被渲染成一个 `<a>` 标签 -->
      <router-link to="/">Go to Foo</router-link>
      <router-link to="/find">Go to Bar</router-link>
    </p>
    <!-- 路由出口 -->
    <!-- 路由匹配到的组件将渲染在这里 -->
    <router-view></router-view>
  </div>
</template>

Find.vue

<template>
  <div>
    <h1>发现音乐页</h1>
    <router-link to="/find/ranking">Go to Ranking</router-link>
    <router-link to="/find/recommend">Go to Recommend</router-link>
    <router-link to="/find/songList">Go to SongList</router-link>
    <!-- 二级路由 -->
    <router-view></router-view>
  </div>
</template>
