# Array数组

# 数组概念

JavaScript 的 Array 对象是用于构造数组的全局对象，数组是类似于列表的高阶对象。

数组是一种类列表对象，它的原型中提供了遍历和修改元素的相关操作。JavaScript 数组的长度和元素类型都是非固定的。因为数组的长度可随时改变，并且其数据在内存中也可以不连续，所以 JavaScript 数组不一定是密集型的，这取决于它的使用方式。一般来说，数组的这些特性会给使用带来方便，但如果这些特性不适用于你的特定使用场景的话，可以考虑使用类型数组 TypedArrayopen in new window。

只能用整数作为数组元素的索引，而不能用字符串。后者称为 关联数组。使用非整数并通过 方括号 或 点号 来访问或设置数组元素时，所操作的并不是数组列表中的元素，而是数组对象的 属性集合上的变量。数组对象的属性和数组元素列表是分开存储的，并且数组的遍历和修改操作也不能作用于这些命名属性。

例子:

var arr = []
arr[0] =1
arr[1] = 4
arr[5] = 6
arr.foo = 'c'
console.log(arr);// [1, 4, 空属性 × 3, 6, foo: 'c']
console.table(arr)
console.log(arr.length);//6

# for...of

# 迭代数组

for...of语句在可迭代对象open in new window（包括 Arrayopen in new window，Mapopen in new window，Setopen in new window，Stringopen in new window，TypedArrayopen in new window，argumentsopen in new window 对象等等）上创建一个迭代循环，调用自定义迭代钩子，并为每个不同属性的值执行语句

let iterable = [10, 20, 30];
for (let value of iterable) {
    value += 1;
    console.log(value);
}
// 11
// 21
// 31

# 迭代Map

let iterable = new Map([["a", 1], ["b", 2], ["c", 3]]);

for (let entry of iterable) {
  console.log(entry);
}
// ["a", 1]
// ["b", 2]
// ["c", 3]

for (let [key, value] of iterable) {
  console.log(value);
}
// 1
// 2
// 3

# 更多关于数组

在 JavaScript 中，数组不是基元open in new window，而是Array具有以下核心特征的对象：

  • JavaScript 数组可以调整大小并且**可以包含不同数据类型open in new window**的混合。（当这些特性不受欢迎时，请改用类型化数组open in new window。）
  • JavaScript 数组不是关联数组，因此不能使用字符串作为索引来访问数组元素open in new window，而必须使用整数作为索引来访问。
  • **JavaScript 数组是零索引open in new window**的：数组的第一个元素在 index 处0，第二个在 index 处1，依此类推——最后一个元素位于数组lengthopen in new window属性 minus的值处1。
  • JavaScript数组复制操作open in new window创建浅拷贝open in new window。（所有带有任何JavaScript 对象的标准内置复制操作都会创建浅拷贝，而不是深拷贝open in new window）。
